# Inventory Management System - Backend Prototype

## Overview
This project serves as a **backend prototype** developed for a client to demonstrate the capabilities of an internal inventory management tool. It is built using **NestJS**, a progressive Node.js framework, ensuring scalability and maintainability.

The primary function of this application is to provide a robust **CRUD (Create, Read, Update, Delete)** interface for managing internal inventory records.

## Features
- **RESTful API**: Standardized endpoints for inventory operations.
- **CRUD Operations**: Full capability to create, view, update, and delete inventory items.
- **Scalable Architecture**: Built on NestJS modules for easy expansion.
- **Prototype Status**: Designed as a proof-of-concept for client review.

## Tech Stack
- **Framework**: [NestJS](https://nestjs.com/)
- **Language**: TypeScript
- **Environment**: Node.js

## Getting Started

### Installation
```bash
$ npm install
```

### Running the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
